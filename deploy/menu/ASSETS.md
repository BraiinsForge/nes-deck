# Menu asset provenance

## Settings cogs

All 36 numbered PNGs from
[Cogwheels and Gears](https://opengameart.org/content/cogwheels-and-gears) by
knekko (Christian Scholz) are retained at their original pixel dimensions in
`uploader/settings-icons/`. The pack is released under CC0 1.0. The web picker
and native dashboard use the same PNG for each `gear-knekko-01` through
`gear-knekko-36` selection. Individual file hashes and dimensions are recorded
in `knekko-settings-icons.tsv`.

The default `gear-steel-outline` remains an adapted monochrome version of
`Medium/15.png`. It was converted at a 25 percent luminance threshold and
reflected across both axes to remove lighting asymmetry. Its original 23 by 23
pixel grid is preserved. The source archive SHA-256 is
`4036273d4917c3f3f13d03ac6a6e1d0db91947b9c04f6a25291c91aec952ffe7` and the
selected PNG SHA-256 is
`63131f81dc3b4b3588416b0d0d496dafd6bd71132019cdddb1d8c33b554637ed`.
